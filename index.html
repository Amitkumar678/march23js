LOOPS

Loops can execute block of code a number of times loops are handle if we want to run same code again and again each time which diffrent value us loops

  • FOR LOOPS
  • WHILE LOOPS
  • DO WHILE LOOPS

for loop

        for(var i=0;i<=10;i++){
            console.log(i)
        }

for loop two type forword loop and backword loop

forword loop

        for(var i=1;i<=10;i++){
            console.log(2+"X"+i+"="+2*i)
        }
        }

BACKWORD LOOP

            for(var i=0;i<=10;--){
                console.log(i)
            }

while loop

when you dont know extaly how many number of timethe loop is going to be executed then will do use while loop

 
                var i=1;
                while(i<10){
                    console.log(i);
                    i++
                }
